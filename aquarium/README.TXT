This enclosing package contains:
- Aquarium, the class that manages and fills the aquarium.
  You should add instructions to fill the aquarium with your creatures here.
- aquarium.creatures, the package holding the things in your aquarium
- aquarium.images, the package (folder) with images you intend to use for your aquarium
  ImageCreature expects this package to have your images in it
  
- aquarium.ui, the package containing UI classes. You don't need to modify this.

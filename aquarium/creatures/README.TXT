- Creature provides a basic interface for all Creatures.
- ImageCreature is an incomplete Creature implementation that supports using a picture file on disk
  as the Creature's appearance.
- LineSwimmer is an almost-complete ImageCreature that always moves in a straight integer-slope line
  and bounces off the sides.
- BoringFish is a LineSwimmer with slope 10 and the image "BoringFish.jpg" which starts in the center
  of the aquarium.

Run AquariumMain to start the aquarium. You probably cannot use BlueJ directly to do this.

Either use the command line, or double-click run.bat in your APAquarium folder.

For example:
C:\Users\Barnett\Documents\BlueJ\APAquarium>java AquariumMain
--or--
C:\Users\Barnett\Documents\BlueJ\APAquarium>run
--or--
Use Windows Explorer to go to APAquarium, then double-click "run" or "run.bat"
